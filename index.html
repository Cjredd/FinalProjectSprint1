BAIS:3500 - Data Mining

This course was designed to introduce students to the fundamentals of data mining and the practical applications of various techniques in data mining. It covers topics such as data preprocessing, data warehousing, clustering, classification, and prediction. We often applied these techniques using popular tools such as Python and R.

BAIS:3020 - Computational Thinking

This course teaches students how to approach complex problems and break them down into smaller, more manageable parts using algorithms and logical reasoning. The course covers topics such as abstraction, decomposition, pattern recognition, and algorithm design. We learned how to apply these concepts using programming languages such as Python and Java.

BAIS:3250 - Data Wrangling

"Data wrangling" encompasses collecting, cleaning, transforming, integrating, and describing data. It is estimated that data scientists spend up to 80% of their time on data wrangling tasks. In addition to being the most time-consuming step of most analytics workflows, wrangling is critical, as it impacts the outcome of all subsequent experiments/analyses. This course introduced essential R programming skills for data wrangling.

BAIS:3140 - Information Visualization

Exposes students to practices that help present complex data in an understandable and meaningful way. The course covers topics such as data visualization principles, perception and cognition, visual design, and interactive visualization techniques. The visualization tool used predominately in this class was Tableau. However, we were also briefly introduced to Microsoft Power BI.
